min25筛
这个算法就是把“一个数最多只有一个大于 n\sqrt nn​ 的质因子”和“⌊nx⌋\lfloor\frac{n}{x}\rfloor⌊xn​⌋ 的值个数 O(n)O(\sqrt n)O(n​)”运用到了极致。

考虑把所有数分成质数和合数。

第一步要求质数的函数值求和。
考虑筛法，拿质数 pip_ipi​ 去筛掉mindiv为 pip_ipi​ 的数。因为 nnn 内不存在一个数的 mindiv 大于 n\sqrt nn​，于是只要用小于等于 n\sqrt nn​ 的质数去筛就好了。

第二步是合数，对于合数枚举它的mindiv和它的指数，即 pep^epe，再除掉递归求解就ok。

#include<bits/stdc++.h>
#define ULL unsigned long long
using namespace std;
const int N=2e6+50;
int t,sq,p[N],tot,sp[N],tp;ULL a[N],mx,n,k,d0[N],st[N],b[N];bool mark[N];
int getid(ULL x){return x<=sq?x:sq+n/x;}//经典trick求id
ULL S(ULL n,int y){//函数意思是2到n mindiv>p[y]的函数值和
    if(n<=p[y])return 0;
    int id=getid(n);ULL ret=(k+1)*(d0[id]-sp[y]);//这里是质数的函数值
    for(int i=y+1;i<=tot&&1ll*p[i]*p[i]<=n;i++){//合数的mindiv一定<=sqrt(n)。这里和下面的剪枝保证了复杂度，一定要加上。
        for(ULL j=p[i],d=k+1;j<=n;j*=p[i],d+=k)
            ret+=d*(S(n/j,i)+(j!=p[i]));//p^k本身也是合数，也要加上。
    }
    return ret;
}
void solve(){
    sq=sqrt(n);tp=0;
    for(ULL l=1,r;l<=n;l=r+1){
        ULL d=n/l;r=n/d;
        st[++tp]=d;d0[getid(d)]=d-1;//先把1减掉，之后好处理。（否则会出现本来想构造合数但是一个质数*1还是质数的情况）
    }
    for(int i=1;i<=tot;i++){
        ULL sq=1ll*p[i]*p[i];if(sq>n)break;
        for(int j=1;st[j]>=sq;j++){//st数组里的数是递减的。被筛掉的数一定>=sq。
            int id=getid(st[j]),iid=getid(st[j]/p[i]);
            d0[id]-=d0[iid]-sp[i-1];//这里减去筛掉的部分，因为数组的值还包括已经是质数的，所以还要减掉。
        }
    }
    printf("%llu\n",S(n,0)+1);//最后把1加上
}
int main(){
    scanf("%d",&t);
    for(int i=1;i<=t;i++)scanf("%llu%llu",&a[i],&b[i]),mx=max(mx,a[i]);
    sq=sqrt(mx);
    for(int i=2;i<=sq;i++){//这里是线性筛预处理。
        if(!mark[i])p[++tot]=i;
        for(int j=1,y;j<=tot&&(y=p[j]*i)<=sq&&(mark[y]=1);j++)
            if(i%p[j]==0)break;
    }
    for(int i=1;i<=tot;i++)sp[i]=i;//质数的函数值前缀和
    for(int i=1;i<=t;i++)n=a[i],k=b[i],solve();
    return 0;
}

min25筛非常好用。只要发现函数值是一个积性函数，pkp^kpk 处的函数值容易计算， ppp 处的函数值是低次多项式即可快速计算。

刚刚的代码是求 ∑i=1nσ0(ik)\sum_{i=1}^n\sigma_0(i^k)∑i=1n​σ0​(ik)。

当 k=2k=2k=2 时有一个疯狂推式子的做法。简单提一提

对 n2n^2n2 约数的每个质因子进行观察可以发现它们的次数要么是奇数要么是偶数（。。。），可以考虑构造映射，对奇数次数的质因子的次数都加1，就变成了一个平方数 d2d^2d2 。d2d^2d2 一定是 n2n^2n2 的约数，于是 ddd 是 nnn 的约数。所以我们可以枚举 nnn 的所有约数 ddd，再对 ddd 的所有不同的质因子选择是否给它的次数 −1-1−1。我们有了一个式子

σ0(n2)=∑d∣n2w(d)\sigma_0(n^2)=\sum_{d|n}2^{w(d)} σ0​(n2)=d∣n∑​2w(d)

其中 w(n)w(n)w(n) 表示 nnn 的不同质因子个数。
再从刚才提到的“对 ddd 的所有不同的质因子选择是否给它的次数 −1-1−1”可以发现我们相当于是对 nnn 的所有没有相同质因子的因数计了个数，即

2w(n)=∑d∣nμ2(d)2^{w(n)}=\sum_{d|n}\mu^2(d) 2w(n)=d∣n∑​μ2(d)

我们现在求前缀和。

∑i=1nσ0(i2)=∑d=1n2w(d)⌊nd⌋\sum_{i=1}^n\sigma_0(i^2)=\sum_{d=1}^n2^{w(d)}\lfloor\frac{n}{d}\rfloor i=1∑n​σ0​(i2)=d=1∑n​2w(d)⌊dn​⌋

我们除法分块以后要求 2w(n)2^{w(n)}2w(n) 的前缀和。

∑i=1n2w(i)=∑d=1nμ2(d)⌊nd⌋\sum_{i=1}^n2^{w(i)}=\sum_{d=1}^n\mu^2(d)\lfloor\frac{n}{d}\rfloor i=1∑n​2w(i)=d=1∑n​μ2(d)⌊dn​⌋

又要求 μ2(n)\mu^2(n)μ2(n) 的前缀和，而这是一个经典的容斥。

∑i=1nμ2(i)=∑i=1nμ(i)⌊ni2⌋\sum_{i=1}^n\mu^2(i)=\sum_{i=1}^{\sqrt n}\mu(i)\lfloor\frac{n}{i^2}\rfloor i=1∑n​μ2(i)=i=1∑n​​μ(i)⌊i2n​⌋

发现这三个式子都是根号计算，于是统统筛出 n23n^{\frac{2}{3}}n32​ 的值之后暴力计算时空复杂度都为 O(n23)O(n^{\frac{2}{3}})O(n32​)。这题太草了。。还卡空间。

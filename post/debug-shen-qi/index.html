2020-02-22

背过背过！每次敲代码先敲一个这个！（看了一下yjc神犇的代码发现的QAQ）

template<class T>void err(const T &x,const char c='\n') {cerr<<x<<c;}
template<class T,class ...Args>void err(const T &x,const Args &...args) {err(x,' '),err(args...);}

使用方法：

#include<bits/stdc++.h>
using namespace std;
template<class T>void err(const T &x,const char c='\n') {cerr<<x<<c;}
template<class T,class ...Args>void err(const T &x,const Args &...args) {err(x,' '),err(args...);}
int main(){
    int a=1,b=2,c=3,d=4;
    err(a,b,c,d);
	err(a,b,c,d);
    return 0;
}

输出：

1 2 3 4
1 2 3 4

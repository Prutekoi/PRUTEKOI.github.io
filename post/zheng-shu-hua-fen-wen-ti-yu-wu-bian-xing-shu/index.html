ϕ(x)=∏i=1∞(1−xi)\phi(x)=\prod_{i=1}^{\infty}(1-x^i) ϕ(x)=i=1∏∞​(1−xi)

欧拉函数与整数划分数的关系

欧拉函数的倒数是整数划分数的生成函数。
即

1ϕ(x)=∑i=0∞p(i)xi\frac{1}{\phi(x)}=\sum_{i=0}^{\infty}p(i)x^i ϕ(x)1​=i=0∑∞​p(i)xi

其中 p(i)p(i)p(i) 为 iii 划分为若干个正整数之和的方案数。
考虑 p(i)p(i)p(i) 的生成函数

∑i=0∞p(i)xi\sum_{i=0}^{\infty}p(i)x^i i=0∑∞​p(i)xi

这个是把所有数的所有的划分方式求了和。于是可以换一种统计方式

∏i=1∞(1+xi+x2i+… )\prod_{i=1}^{\infty}(1+x^i+x^{2i}+\dots) i=1∏∞​(1+xi+x2i+…)

化一下等比数列就会发现是欧拉函数的倒数。

我们考虑有两种dp的方式。
1.dp[i][j]dp[i][j]dp[i][j] 表示用的数都小于等于 iii 和为 jjj 的方案数。这是个完全背包模型。
2.dp[i][j]dp[i][j]dp[i][j] 表示 iii 个数和为 jjj 的方案数。这是通常写的。

我们发现第一个dp状态量与数大小有关，第二个dp状态量与数个数有关。那么根号平衡一下。

我们把数分成小于等于 n\sqrt nn​ 和大于 n\sqrt nn​ 两部分，分别求出只用这两部分的数的方案数。然后再枚举两部分的和都为多少，乘法原理一下就好了。

这样小于等于 n\sqrt nn​ 的部分用1方式求解，状态量 nnn\sqrt nnn​，大于 n\sqrt nn​ 的部分用2方式求解，状态量 nnn\sqrt nnn​，于是完美解决。注意第二个dp时要滚动数组。

#include<bits/stdc++.h>
using namespace std;
const int N=2e5+50,mod=1e9+7;
int n,sq,f[N],g[N],h[N],gg[N],ans;
int main(){
    scanf("%d%d",&n);sq=sqrt(n);
    f[0]=1;
    for(int i=1;i<=sq;i++)for(int j=i;j<=n;j++)(f[j]+=f[j-i])%=mod;
    g[0]=gg[0]=1;
    for(int i=1;i<=sq;i++){
        memcpy(h,g,sizeof(g));
        memset(g,0,sizeof(g));
        for(int j=i*(sq+1);j<=n;j++)
            g[j]=(h[j-sq-1]+g[j-i])%mod,
            (gg[j]+=g[j])%=mod;
    }
    for(int i=0;i<=n;i++)ans=(ans+1ll*f[i]*gg[n-i])%mod;
    printf("%d\n",ans);
    return 0;
}
